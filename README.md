# Private Django Project #

趣味で作っているDjangoアプリケーションを公開していく．

### Requirement ###

このプロジェクトを運用する際は以下のアプリケーション・ライブラリが必要になります．

- Python 2.7

- pip

Python, pip をインストール後，requirements.txt を使いライブラリのインストールを行って下さい．

    pip install -r requirements.txt
